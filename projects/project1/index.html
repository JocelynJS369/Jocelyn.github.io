    超视距抛洒物感知预警系统

    2025-07-262025-08-28

    基于车联网的超视距抛洒物感知预警系统

    2024-05-06
    学科竞赛项目

    项目介绍

    提出一种基于车联网的超视距抛洒物检测系统，系统基于车联网（V2X）系统搭建，由车端和路端对高速公路抛洒物进行协同感知，基于 C-V2X 通信技术实现"车-车、车-路"协同预警。

    主要工作

    大创负责人 + 路演人

    负责大创项目的整体设计和部分技术实现

    具体职责：

    • 负责Web界面搭建(Hbuilder)
    • 参与抛洒物数据集标注制作(labelimg)
    • 负责调试部分算法(PP-YOLOE)
    • 参与路侧感知站数据采集

    技术栈：

    类别 详情
    编程语言 Python
    深度学习框架 PyTorch
    算法 YOLOv9、YOLOv8、PP-YOLOE、DETR
    硬件 松灵SCOUT MINI Pro 小车，PIX自动驾驶底盘，路侧感知站
